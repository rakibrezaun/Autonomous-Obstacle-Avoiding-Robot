# Autonomous-Obstacle-Avoiding-Robot
Design and Implementation of an Autonomous Obstacle-Avoiding Robot with 180° Ultrasonic Scanning and High-Discharge Power System

This project details the engineering process behind an autonomous 3WD robot designed to navigate unknown environments. The system integrates an Arduino Uno microcontroller with an HC-SR04 ultrasonic sensor mounted on a servo motor to perform spatial scanning. Significant focus was placed on power optimization; the project transitioned from a standard 9V supply to a 3S (11.1V) Li-ion configuration to overcome L298N voltage drops and ensure system stability. Experimental results demonstrate a robust "Scan-and-Compare" navigation logic that effectively mitigates collision risks.
